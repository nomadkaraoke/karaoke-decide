# Plan Command

Create a detailed implementation plan for a feature or task.

## Instructions

1. Understand the requirements thoroughly
2. Research existing code patterns in the codebase
3. Break down into implementable steps
4. Consider edge cases and error handling
5. Write the plan to `.claude/plans/<feature-name>.plan.md`

## Output Format

```markdown
# Plan: <Feature Name>

## Overview
Brief description of what we're building

## Requirements
- List of functional requirements
- Non-functional requirements

## Technical Approach
How we'll implement this

## Implementation Steps
1. Step 1
2. Step 2
3. ...

## Files to Create/Modify
- path/to/file.py - description

## Testing Strategy
How we'll test this

## Open Questions
Any uncertainties to resolve
```
