# Implement Command

Implement a feature based on an existing plan.

## Instructions

1. Read the plan from `.claude/plans/` if available
2. Follow the implementation steps in order
3. Write tests for each component
4. Update documentation as needed
5. Run tests to verify functionality

## Checklist

- [ ] Read and understand the plan
- [ ] Implement core functionality
- [ ] Add error handling
- [ ] Write unit tests
- [ ] Update API documentation if applicable
- [ ] Run `make test` to verify
- [ ] Update docs/README.md status if needed
